# 🔗 URL Shortener

A simple and clean URL shortener web app built with **Node.js**, **Express**, **MongoDB**, and **EJS**. It lets you paste any long URL and get a short, shareable link — and tracks how many times each link has been clicked.

---

## 🚀 Features

- **Shorten URLs** — Generate a unique short ID for any long URL
- **Redirect** — Visiting the short URL instantly redirects to the original
- **Click Tracking** — Every visit is recorded with a timestamp in the database
- **Analytics** — View total clicks and full visit history per short URL
- **Homepage Table** — See all generated URLs along with their click counts

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or above)
- [MongoDB](https://www.mongodb.com/) running locally on port `27017`

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/dikshaaaraj/URL_shortener_project.git
cd URL_shortener_project
```

**2. Install dependencies**
```bash
npm install
```

**3. Start MongoDB** (if not already running)
```bash
mongod
```

**4. Start the server**
```bash
node index.js
```

Or with auto-reload using nodemon:
```bash
npx nodemon index.js
```

**5. Open in browser**
```
http://localhost:8001
```

## 📌 API Routes

| Method | Route | Description |
|---|---|---|
| `GET` | `/` | Homepage — lists all shortened URLs |
| `GET` | `/test` | Test route — same as homepage |
| `POST` | `/url` | Generate a new short URL |
| `GET` | `/:shortId` | Redirect to original URL |
| `GET` | `/url/analytics/:shortId` | Get click count and visit history |

### Example — Generate a Short URL

**POST** `/url`

Request body (form data):
```
url=https://www.example.com
```

Response: renders homepage with the generated short ID.

### Example — Analytics

**GET** `/url/analytics/abc123`

```json
{
  "totalClicks": 5,
  "analytics": [
    { "timestamp": 1714203600000 },
    { "timestamp": 1714207200000 }
  ]
}
```

## 🗄️ Database Schema

**Collection: `urls`**

| Field | Type | Description |
|---|---|---|
| `shortId` | String | Unique short identifier |
| `redirectURL` | String | Original long URL |
| `visitHistory` | Array | List of `{ timestamp }` objects |
| `createdAt` | Date | Auto-generated by Mongoose |
| `updatedAt` | Date | Auto-generated by Mongoose |
